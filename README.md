### Proyecto final de Desarrollo de Software

* En este proyecto esta anexado el front aparte y el backend aparte, adicional se tiene actions unos test y pruebas que se ejecutan al iniciar o subir en un repositorio.

* El front esta en la carpeta Nova-master.
* El backend es todo el contenido que esta como tal en el repositorio.
